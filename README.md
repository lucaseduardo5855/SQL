# 🗄️ Estudo SQL

> Repositório destinado para aulas, treinamentos e exercícios práticos de Banco de Dados utilizando SQL.

## 🎯 Objetivo
Este espaço foi criado para documentar a minha evolução no aprendizado de banco de dados, armazenando as queries, testes e desafios que resolvo durante os meus estudos de Back-end.

## 🚀 Tecnologias e Ferramentas Utilizadas
* **SQL** (Linguagem de consulta estruturada)
* **SQLite** (Motor de banco de dados)
* **VS Code** (Ambiente de desenvolvimento)
* **Git & GitHub** (Versionamento de código)

## 📁 Estrutura do Repositório

Conforme avanço nos estudos, os módulos vão sendo separados por pastas:

* **📂 01/**: Fundamentos básicos. 
  * Consultas com `SELECT` e criação de colunas dinâmicas (Apelidos/Aliases).
  * Filtros com `WHERE`, `IN` e `BETWEEN`.
  * Manipulação de datas (`datetime`, `strftime`).
  * Lógica condicional com `CASE WHEN`.
